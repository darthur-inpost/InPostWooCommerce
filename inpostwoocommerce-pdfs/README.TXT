Make sure this folder has 777 permisions for the local saving of PDF files.
Or that the Apache www-data user or equivalent has write access to the folder.

This folder must be moved to be at the SAME level as the InPost plugin, e.g.
/root/folder/wp-content/plugins/inpostwoocommerce
/root/folder/wp-content/plugins/inpostwoocommerce-pdfs
